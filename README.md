# LaLiga Stats Analysis

## Project Overview
This project focuses on analyzing LaLiga football statistics to derive meaningful insights. The analysis is performed using Python, and the data is sourced from CSV files containing LaLiga statistics.

## Files in the Repository
- **laliga_stats.csv**: The raw dataset containing LaLiga statistics.
- **cleaned_laliga_stats.csv**: The cleaned and preprocessed dataset.
- **laliga_draft.py**: The Python script used for data analysis.

## How to Run the Project
1. Ensure you have Python installed on your system.
2. Install any required dependencies (if applicable).
3. Run the script using the following command:
   ```bash
   python laliga_draft.py
   ```

## Project Flow

1. **Data Collection**:

2. **Data Cleaning**:

3. **Data Analysis**:

## Dashboard Details

The dashboard provides an interactive way to explore the LaLiga statistics. Below are the key features of the dashboard:

- **Overview Section**:
  - Displays key metrics such as total goals, matches played, and top-performing teams.

- **Team Performance Analysis**:
  - Allows users to filter and compare team statistics.
  - Includes metrics like goals scored, goals conceded, and win rates.

- **Player Insights**:
  - Highlights top players based on goals, assists, and other performance indicators.

- **Trends and Visualizations**:
  - Provides time-series graphs and other visualizations to analyze trends over seasons.

- **Custom Filters**:
  - Users can apply filters based on seasons, teams, and players to customize their view.
